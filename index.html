ESP32 Wi-Fi Provisioner

Use this page to send a Wi-Fi network name (SSID) and password to an ESP32 board that is plugged into this computer over USB. The values are sent only to the selected serial device. Nothing is uploaded to the internet.

Your browser does not support Web Serial. Please use a recent version of Google Chrome or Microsoft Edge on a desktop computer, and open this page over https:// or from http://localhost.

Step 1 of 3: Plug in the ESP32

Connect the ESP32 board to this computer with its USB cable. Close any other program that might be using the board, such as the Arduino Serial Monitor.

    How this works (for curious users and developers)
    

      This page uses the Web Serial API to talk to the ESP32 at
      115200 baud, 8 data bits, no parity, 1 stop bit. It speaks a small text
      protocol:
    

    
    
      
  • Browser sends <<PROV?>> repeatedly to get the board's attention.
    • 
      
  • ESP32 replies <<PROV!>> when ready.
    • 
      
  • ESP32 may also send <<PROV:ID name_b64>> with a human-readable device name.
    • 
      
  • Browser sends <<PROV:SET ssid_b64 pass_b64 crc16>>.
    • 
      
  • ESP32 replies <<PROV:OK>> on success or <<PROV:ERR reason>> on failure.
    • 
    

    

      The SSID and password are Base64 encoded so any characters are transmitted
      safely, and a CRC-16/CCITT-FALSE checksum protects against line noise.
